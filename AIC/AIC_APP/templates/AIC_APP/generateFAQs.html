User Guide

When To Use ?

This module is useful if you already have structured FAQs in the form of multiple Questions-Answers pair.

After submitting FAQs, our model will paraphrase each questions to the different forms in which end user can ask one single FAQ.

You will be given chance to evaluate paraphrased FAQs with answers once. You can even download this as a file and then preview it. One can even delete FAQs if he/she finds it irrelevant.

After evaluating paraphrased FAQs, one can train the ML model.
