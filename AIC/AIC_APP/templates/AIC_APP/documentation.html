About us

Your bot is a platfrom which allows you to build your own chat assistant as well as you can use our API to integrate it with your website / mobile App

How To Use?

You need to follow this steps to use our API

  1. Provide your data and Generate FAQs
  2. Review Generated FAQs
  3. Train model for your chat assistant
  4. Use API key to export

Generate API Key

Step 1. Go To Your Profile

Step 2. Go to Manage API

Step 3. Generate your API Key

How to use API Key?

Step 1. Activate Your API Key
Step 2. Use Following url to fetch response
http://localhost:8000/AIC/api/chat_assistant/?api=<Your Api Key>&query=<Your Question>
Example
REQUEST : GET http://localhost:8000/AIC/api/chat_assistant/?api=abcdefghijklmnopqurstvbitnre&query=What is AICTE?

Note : Before using API Key please ensure that it is activated. Do not share your API key with anyone.

Types of Response

1. Query resolved successfully

Response Code : 200 OK

{
    "status": "Success",
    "message": < Response For Your Question >
}
2. Insufficient Data to answer question

Response Code : 404 Not Found

{
    "status": "Failed",
    "message": < Your Error Message >
}

Note : This might occure to insufficient or incomplete data or the query you asked is not found. if you won't set error message default error message "Sorry We Can't Help You" will be used.

3. You didn't send any api key or wrong API key

Response Code : 400 Bad Request

{
    "status": "Failed",
    "message": "Please Provide Active API Key"
}
4. You didn't send any query

Response Code : 400 Bad Request

{
    "status": "Failed",
    "message": "Query is Required"
}

Request Parameters

Sr No. Parameter Significance
1 api API Key of user
2 query Question of user
3 source Language of sender
4 source Language of Response
